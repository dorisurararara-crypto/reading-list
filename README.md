# Reading List — 무료 Pocket 대체

Pocket 종료 발표(2025년 7월 종료)에 대비한 가벼운 PWA. 가입 없이 URL을 모았다가 나중에 읽습니다.

## 특징
- **무료 / 광고 없음 / 계정 없음**
- **모든 데이터는 기기 localStorage에만** 저장 (서버·추적 없음)
- **PWA**: 모바일에서 "홈 화면에 추가" 시 앱처럼 동작
- **오프라인** 동작 (서비스워커 캐시)
- **공유 받기** (PWA Share Target): 다른 앱에서 "공유 → Reading List" → 자동 추가
- **JSON 백업/복원**

## 사용
- 데모: https://dorisurararara-crypto.github.io/reading-list/
- 모바일에서 사파리/크롬 → "홈 화면에 추가" → 앱처럼 사용
- URL 붙여넣기 → 추가 → 자동으로 제목/요약 가져옴 (r.jina.ai 무료 프록시)
