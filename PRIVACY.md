# 개인정보 처리방침 (Privacy Policy)

**최종 수정일**: 2026-05-04

## 한국어

Reading List는 사용자의 개인정보를 수집하지 않으며 외부 서버로 전송하지 않습니다.

### 수집·저장 정보
- 사용자가 저장한 URL, 제목, 요약, 즐겨찾기·읽음 표시 상태
- 위 정보는 사용자의 브라우저 `chrome.storage.local` (확장 사용 시) 또는 `localStorage` (PWA 사용 시) 에만 저장됩니다.
- 우리는 어떠한 서버도 운영하지 않으며, 위 정보는 절대 외부로 전송되지 않습니다.

### 외부 서비스
- 페이지 제목·요약 자동 추출을 위해 `r.jina.ai` 무료 공개 API에 사용자가 저장하려는 URL 1건을 전송합니다. 이는 사용자가 명시적으로 페이지를 저장할 때만 발생하며, 응답으로 받는 데이터는 텍스트뿐입니다.

### 권한 사용
- `storage` — 저장 데이터 보관 (로컬)
- `contextMenus` — 우클릭 메뉴 제공
- `activeTab` / `tabs` — 사용자가 액션 실행 시 현재 탭 URL·제목 읽기
- `notifications` — 저장 결과 알림

### 제3자 공유·판매
- 일체 없음. Google Analytics 등 추적 도구 사용하지 않습니다.

### 어린이 개인정보
- 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다.

### 변경 사항
- 본 방침이 변경되는 경우 본 페이지에서 공지합니다.

### 문의
- GitHub Issues: https://github.com/dorisurararara-crypto/reading-list/issues

## English

Reading List does not collect or transmit any personal data to external servers.

### Stored Data
- URLs, titles, descriptions, favorite/archived states of items the user saves.
- All data is stored exclusively in the user's browser `chrome.storage.local` (extension) or `localStorage` (PWA).
- We operate no servers and never transmit your data externally.

### External Services
- For automatic title/description extraction, the URL the user is saving is sent to `r.jina.ai` (free public API) at the moment of saving. The response is text only.

### Permissions
- `storage` — local data persistence
- `contextMenus` — right-click menu
- `activeTab` / `tabs` — read current tab URL/title only when user invokes save action
- `notifications` — show save results

### Third-Party Sharing / Sale
- None. No analytics or tracking.

### Children's Privacy
- We do not knowingly collect personal information from children under 13.
